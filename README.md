# BioML Experiment 1

Computational biology experiments using [LabDAO PLEX](https://docs.labdao.xyz/) — a CLI for running scientific workflows on peer-to-peer compute.

## Tools Used
- **LabDAO PLEX** — job orchestration on Bacalhau P2P compute network
- **Jellyfin** — GPU-accelerated video encoding backend
- **ColabDesign / RFDiffusion / ProteinMPNN** — protein binder design
- **ColabFold** — protein complex simulation

## Workflows
- Protein binder design against target structures (`.pdb`)
- Protein complex simulation from sequences
